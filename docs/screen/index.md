## 画面仕様書の書き方

各画面仕様書には以下を記載する:

1. **概要** — 画面の目的と役割
2. **URL / ルート** — axum のエンドポイント
3. **レイアウト** — 画面構成の説明
4. **操作仕様** — ユーザー操作とその結果
5. **API連携** — 使用する API エンドポイント
6. **エラー処理** — エラー時の表示・動作
