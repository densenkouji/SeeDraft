# メイン画面

## 概要

アプリ起動時に表示されるメイン画面。axum が配信する HTML を Tauri ウィンドウで表示する。

## URL / ルート

- `GET /index` — メイン画面の HTML を返す

## レイアウト

- ページ中央にタイトルと説明文を配置
- レスポンシブ対応 (CSS Grid)

## 操作仕様

| 操作 | 結果 |
|------|------|
| アプリ起動 | axum サーバー起動後、自動的にこの画面を表示 |
| ウィンドウ閉じる | axum サーバーを graceful shutdown して終了 |

## API連携

なし (静的 HTML を直接返却)

## エラー処理

- サーバー起動タイムアウト (10秒) 時はアプリ終了
- ポート競合時はエラーメッセージを出力して終了
