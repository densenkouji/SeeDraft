## API 仕様書の書き方

各 API 仕様書には以下を記載する:

1. **エンドポイント** — メソッド、パス
2. **リクエスト** — パラメータ、ヘッダ、ボディ
3. **レスポンス** — ステータスコード、ボディ
4. **エラー** — エラーレスポンスの形式
5. **実装場所** — 対応する Rust のハンドラ関数
